# Tribuna F1 - Landing Page

## Archivos
- `index.html` — página principal con el hero y contenido introductorio.
- `style.css` — estilos minimalistas, responsivos y accesibles.

## Integración
1. Copia `index.html` y `style.css` al directorio de tu proyecto.
2. Asegúrate de conservar los enlaces CDN en el `<head>` de `index.html`:
   - Normalize CSS
   - Google Fonts (Poppins + Inter)
   - Font Awesome
3. Abre `index.html` en el navegador.

## Personalización rápida
- Cambia la imagen del hero reemplazando la URL en `background-image` dentro de `style.css`.
- Actualiza los enlaces del CTA primario o secundario directamente en `index.html`.

## Notas
- Usa un solo `<main>` en la página y estructura semántica clara.
- El `hero` está optimizado para móviles y mantiene contraste alto para accesibilidad.
- El enlace a Instagram es un ejemplo; reemplázalo por la cuenta real.
